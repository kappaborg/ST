Shoe Store System

SE302 Homework 01 - Task 1: Equivalence Class Testing

System Requirements

  • Item name: Alphabetic characters only, length 2-15 characters
  • Sizes: Whole numbers between 26-55
  • Format: itemName,size1,size2,...
  • Sizes must be in ascending order
  • Maximum of 5 sizes per item
  • Commas used as separators
  • Blank spaces are ignored

Equivalence Classes Reference

Valid Classes (EC1-EC12)

EC1: Name length = 2
EC2: Name length 3-14
EC3: Name length = 15
EC4: Size = 26 (min)
EC5: Size 27-54
EC6: Size = 55 (max)
EC7: Single size
EC8: 2-4 sizes
EC9: 5 sizes (max)
EC10: Ascending order
EC11: Proper comma
EC12: Spaces around commas

Invalid Classes (EC13-EC28)

EC13: Name < 2 chars
EC14: Name > 15 chars
EC15: Name has numbers
EC16: Name has special chars
EC17: Empty name
EC18: Size < 26
EC19: Size > 55
EC20: Size has decimal
EC21: Size has letters
EC22: > 5 sizes
EC23: No sizes
EC24: Not ascending
EC25: Missing comma
EC26: Missing comma between sizes
EC27: Extra comma
EC28: Empty line
